# meme-maker

Скилл для Claude Code, который превращает сырую идею в готовый мем-картинку с русскими надписями. Подбирает подходящий шаблон (Drake, Distracted Boyfriend, This is Fine, Galaxy Brain, Two Buttons, Spongebob, DiCaprio Cheers и ещё 18+), формулирует панчи и рисует через **Nano Banana 2 (Google Gemini)** по MCP-серверу.

## Что делает

1. Парсит идею (противопоставление, количество элементов, эмоциональный вектор).
2. Подбирает 5–7 мем-шаблонов под структуру идеи.
3. Сразу формулирует **готовые русские подписи** к каждому шаблону — не плейсхолдеры.
4. Показывает список, ждёт твоего выбора.
5. Рисует выбранные мемы параллельно через MCP `nanobanana`.
6. Поддерживает итерацию — перегенерация, точечная правка текста через `gemini_edit_image`.

## Установка

### 1. Клонировать скилл

В папку Claude Code-скиллов (`~/.claude/skills/` для глобальных или `.claude/skills/` в репо проекта):

```bash
cd ~/.claude/skills
git clone <URL_ЭТОГО_РЕПО> meme-maker
```

### 2. Подключить MCP-сервер `nanobanana`

Скилл использует публичный пакет [`@ycse/nanobanana-mcp`](https://www.npmjs.com/package/@ycse/nanobanana-mcp).

Получи бесплатный API-ключ:

1. Открой [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Залогинься Google-аккаунтом → **Create API key** → скопируй строку вида `AIza...`

Подключи MCP к Claude Code:

```bash
claude mcp add nanobanana --env GOOGLE_AI_API_KEY=ВСТАВЬ_СВОЙ_КЛЮЧ -- npx -y @ycse/nanobanana-mcp@latest
```

Перезапусти Claude Code, чтобы MCP подгрузился.

### 3. Проверка

Запусти Claude Code в любой папке и напиши:

```
/meme-maker сделай мем про то, что я три года на одной модели, а индустрия каждую неделю выкатывает новую
```

Скилл должен предложить 5–7 шаблонов с готовыми подписями. Если предложил — всё работает.

## Опциональный авторский голос-профиль

По умолчанию скилл рисует в **нейтральном режиме** — общечеловеческий юмор, без узких отсылок.

Если у тебя есть свой канал/блог/бренд и ты хочешь, чтобы подписи звучали в твоём голосе:

1. Скопируй шаблон: `cp references/voice_profile.example.md references/voice_profile.md`
2. Заполни под свой тон (тон, лексика, эталонные подписи).
3. Скажи скиллу: «используй мой голосовой профиль» или «в моём голосе».

## Структура

```
meme-maker/
├── SKILL.md                              # Главный файл скилла (читается Claude Code)
├── README.md                             # Этот файл
├── LICENSE                               # MIT
└── references/
    ├── meme_catalog.md                   # Каталог 18+ мем-шаблонов: формат, роли, под какие идеи
    ├── prompting_rules.md                # Правила промптов: английский, Russian Cyrillic, spatial-layout, скелеты
    └── voice_profile.example.md          # Шаблон опционального авторского голоса-профиля
```

## Расширение

Каталог мем-шаблонов **расширяемый**. Если нужного шаблона нет (например, Surprised Pikachu, Disaster Girl, Gru's Plan, Roll Safe) — попроси скилл нарисовать, и после успешной генерации он сам добавит блок в `meme_catalog.md` и промпт-скелет в `prompting_rules.md`.
