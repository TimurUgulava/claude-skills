# Claude Skills

Коллекция кастомных скиллов для [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## Skill Architect

**Архитектор скиллов** — проектирует и собирает промышленные скиллы для Claude Code по 4-фазному протоколу.

### Что делает

Берёт техническое задание (или просто описание задачи словами) и превращает его в готовый скилл: директория с `SKILL.md`, набор `references/` и упакованный `.skill` файл.

### 4 фазы работы

| Фаза | Что происходит |
|------|---------------|
| **Discovery** | Выясняет задачу, триггеры, формат вывода. Задаёт уточняющие вопросы |
| **Blueprint** | Выбирает архитектурный паттерн, рисует Mermaid-схему, предлагает структуру |
| **Compilation** | Собирает SKILL.md + references, упаковывает в `.skill` |
| **Debug** | Если что-то работает не так — предлагает протестировать и докручивает |

### Принципы

- **Атомарная детализация** — каждое требование декомпозируется на проверяемые инструкции
- **SKILL.md ≤ 100 строк** — ядро в body, детали в references. Экономит токены при каждом триггере
- **Архитектор, не исполнитель** — создаёт скилл для задачи, а не выполняет задачу сам
- **Генерализация** — скилл должен работать тысячи раз, не только на примерах из ТЗ

### Установка

Скопируйте директорию `skill-architect/` в ваш проект:

```bash
cp -r skill-architect/ /path/to/your/project/.claude/skills/skill-architect/
```

Или установите из `.skill` файла (если поддерживается вашей версией Claude Code).

### Как использовать

Скажите Claude Code что-то вроде:

- «Создай скилл для анализа PR»
- «Мне нужен скилл, который генерирует отчёты»
- «Спроектируй скилл для code review»

Skill Architect сам проведёт вас через все фазы.

### Структура

```
skill-architect/
├── SKILL.md                              # Ядро скилла
├── references/
│   ├── protocol.md                       # Фазы Discovery и Blueprint
│   ├── compilation-guide.md              # Правила сборки SKILL.md
│   ├── packaging.md                      # Упаковка в .skill
│   ├── debug-guide.md                    # Матрица диагностики
│   ├── patterns.md                       # Архитектурные паттерны
│   ├── skill-template.md                 # Шаблоны SKILL.md
│   └── eval-guide.md                     # Тестирование и оптимизация
└── examples/
    └── example-invoice-skill.md          # Пример готового скилла
```
