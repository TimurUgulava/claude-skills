# Фаза 4: Отладка + User Guide

## Матрица диагностики

Триггер: жалоба на качество, формат, триггеринг.

| Проблема | Сигнал | Исправление |
|----------|--------|-------------|
| Не триггерится | Скилл не загружается | Добавить фразы в description |
| Ложные срабатывания | Активируется для чужих запросов | Добавить negative triggers |
| Игнорирует body | Следует только description | Убрать workflow из description |
| Нестабильный вывод | Разные результаты | Добавить шаблоны, бандлить скрипты |
| Медленный | Раздутый контекст | Детали → references/, body ≤ 100 строк |

## Алгоритм отладки

1. **Диагностика** — определи класс проблемы по таблице
2. **Патч** — напиши исправленный блок, объясни что и *почему* изменилось
3. **Верификация** — «Помогло? Тестируем дальше?»
